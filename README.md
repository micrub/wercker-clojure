# wercker-clojure

A Leiningen template for clojure projects development using Wercker.

## Usage

### Requirments

[Setup wercker cli](http://devcenter.wercker.com/cli/installation/index.html)

### Development stage

In order to run tests in container, execute:

````
wercker dev
```
